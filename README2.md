# GitHub 简介

## 步骤 1：创建分支



*欢迎来到 “GitHub 简介”！👋*

**什么是 GitHub？**： GitHub 是一个使用 *[Git](https://docs.github.com/get-started/quickstart/github-glossary#git)* 进行版本控制的协作平台。GitHub 是共享和贡献[开源](https://docs.github.com/get-started/quickstart/github-glossary#open-source)软件的热门场所。
📺 [视频：什么是 GitHub？](https://www.youtube.com/watch?v=pBy1zgt0XPc)

**什么是存储库？**：*[存储库](https://docs.github.com/get-started/quickstart/github-glossary#repository)*是包含文件和文件夹的项目。存储库跟踪文件和文件夹的版本。有关详细信息，请参阅 GitHub Docs 中的“[关于存储库](https://docs.github.com/en/repositories/creating-and-managing-repositories/about-repositories)”。

**什么是分支？**：*[分支](https://docs.github.com/en/get-started/quickstart/github-glossary#branch)*是存储库的并行版本。默认情况下，您的仓库有一个名为 的分支，它被视为最终分支。通过创建其他分支，您可以复制存储库的分支并安全地进行任何更改，而不会中断主项目。许多人使用分支来处理特定功能，而不会影响项目的任何其他部分。`main``main`

分支允许您将工作与分支分开。换句话说，当你做出贡献时，每个人的工作都是安全的。有关详细信息，请参阅“[关于分支](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-branches)”。`main`

**什么是个人资料 README？**：*[个人资料 README](https://docs.github.com/account-and-profile/setting-up-and-managing-your-github-profile/customizing-your-profile/managing-your-profile-readme)* 本质上是 GitHub 个人资料上的“关于我”部分，您可以在其中与 GitHub.com 上的社区分享有关您自己的信息。GitHub 在个人资料页面顶部显示您的个人资料 README。有关详细信息，请参阅“[管理配置文件 README”。](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-github-profile/customizing-your-profile/managing-your-profile-readme)

### ⌨️ 活动：您的第一个分支



1. 打开新的浏览器选项卡并导航到新创建的存储库。然后，在阅读此选项卡中的说明时，执行第二个选项卡中的步骤。

2. 导航到存储库标题菜单中的 **< > Code** 选项卡。

   ![](C:\Users\22536\Desktop\新建文件夹\code-tab.png)

   

3. 单击 **main** 分支下拉列表。

   ![](C:\Users\22536\Desktop\新建文件夹\main-branch-dropdown.png)

4. 在字段中，为您的分支命名 。在这种情况下，名称必须为 to trigger the course workflow。`my-first-branch``my-first-branch`

5. 单击 **Create branch： my-first-branch** 创建您的分支。

      [![创建分支按钮](https://github.com/Exp-Intro-to-GitHub-Flow-Cohort-2/series-intro-to-github-flow-Song000/raw/main/images/create-branch-button.png)](https://github.com/Exp-Intro-to-GitHub-Flow-Cohort-2/series-intro-to-github-flow-Song000/blob/main/images/create-branch-button.png)

6. 等待大约 20 秒，然后刷新此页面（您正在按照说明操作的页面）。[GitHub Actions](https://docs.github.com/en/actions) 将自动更新到下一步。
